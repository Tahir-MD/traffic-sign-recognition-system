## ✨ Features

- 🚦 Real-time detection
- 📸 Image processing
- 📹 Webcam support
- 📊 Performance analytics
- 🎯 43 traffic sign classes

## 🚀 Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/Tahir-MD/traffic-sign-recognition-system.git
cd traffic-sign-recognition-system

# Install dependencies
pip install -r requirements.txt

# Run application
streamlit run app.py
